
# Pokedex-Com-React

Este é um projeto de Pokedex desenvolvido com React e TypeScript, que permite visualizar informações detalhadas sobre diversos Pokémon. A aplicação faz requisições à PokeAPI para buscar dados em tempo real, oferecendo uma experiência interativa e responsiva.

## Funcionalidades

- Listagem de Pokémon com imagens e informações detalhadas.
- Busca por nome ou número do Pokémon.
- Interface responsiva para dispositivos móveis.
- Navegação fácil entre diferentes gerações de Pokémon.

## Tecnologias Utilizadas

- **React** com **TypeScript** para uma melhor tipagem e manutenção do código.
- **Styled-Components** para estilização modular e reutilizável.
- **Axios** para requisições HTTP à PokeAPI.
- **Vite** para um ambiente de desenvolvimento rápido e moderno.
- **ContextAPI** para um melhor organizando do código e melhorando sua performace.
 
## Como Executar o Projeto

1. Clone o repositório:
    ```bash
    git clone https://github.com/CronusLDX/Pokedex-Com-React.git
    ```
2. Instale as dependências:
    ```bash
    cd Pokedex-Com-React
    npm install
    ```
3. Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4. Acesse a aplicação no navegador:
    ```
    http://localhost:5173
    ```
